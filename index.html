Chrome Built-in AI Showcase

Google Chrome で動作するローカル組み込み AI（Gemini Nano）を活用した、オンデバイス Multimodal Prompt API のインタラクティブ・デモポータル

🖥️

画面キャプチャ・アナライザー

開いているウィンドウを選択してストリーミングキャプチャを行い、オンデバイス AI がコーディングエラーや作業内容を自動で解析します。

定期的自動スクリーンキャプチャ
競合を回避した高速 GPU 推論
解析履歴ログのタイムライン表示
デモを起動する 🚀
🔒

個人情報検出アップローダー

写真をアップロードまたはドラッグ＆ドロップし、ネット公開や送信をする前に、個人情報が含まれていないかをローカルで安全にチェックします。

ドラッグ＆ドロップによる画像解析
伝票や隠れたメモ等の現実的サンプル
サーバーに送らない安心のオンデバイス検知
デモを起動する 🚀

⚙️ デモ動作に必要な Chrome の準備設定

Step 1

Chrome Flags の有効化

chrome://flags を開き、以下を有効にします：
• #optimization-guide-on-device-model (Enabled BypassRamRequirement)
• #prompt-api-for-gemini-nano (Enabled)

Step 2

Gemini Nano モデルのダウンロード

chrome://components を開き、
• Optimization Guide On Device Model の「アップデートを確認」をクリックして、モデル(約3GB)のダウンロードを完了させてください。
