🛡️ Chrome 組み込み AI: 個人情報チェッカー

Multimodal Prompt API を使用した、アップロード前の個人情報検知デモ (オンデバイス処理)

AI機能を確認中...

⚠️ Chrome 組み込み AI (Gemini Nano) が利用できません

このデモを実行するには、Chrome のオンデバイス AI (Multimodal Prompt API) が有効である必要があります。以下の設定を確認してください：

  1. Chrome 148 以上を使用していることを確認してください。
  2. chrome://flags にアクセスし、以下を設定してください：
    • #optimization-guide-on-device-model を "Enabled BypassRamRequirement" または "Enabled" に設定
    • #prompt-api-for-gemini-nano を "Enabled" に設定
  3. 設定変更後、ブラウザを再起動し、chrome://components を開いて Optimization Guide On Device Model の「アップデートを確認」をクリックし、モデルのダウンロードを完了させてください。
